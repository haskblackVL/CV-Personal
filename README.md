# CV Personal - Víctor Daniel Luna Juárez

Este es el proyecto del Curriculum Vitae profesional de **Víctor Daniel Luna Juárez**, Especialista en Logística. El sitio ha sido optimizado con un diseño responsivo de primer nivel, elegante y corporativo, siguiendo mejores prácticas de desarrollo web y diseño moderno.

## Mejoras Implementadas

1. **Diseño Responsivo:** 
   - Se reestructuraron las columnas fijas utilizando Grid y Flexbox.
   - El CV se adapta perfectamente a dispositivos móviles y tabletas mediante media queries (`@media (max-width: 850px)`), acomodando la barra lateral arriba del contenido principal de manera fluida.

2. **Resolución de Deformación de Imagen:**
   - La foto de perfil se ha configurado con dimensiones fijas proporcionales (`150px` x `150px`) y la propiedad `object-fit: cover`. Esto garantiza que el rostro mantenga su aspecto circular sin deformarse al cambiar de pantalla.

3. **Unificación de Estructura (Solución de Doble Contenedor):**
   - Se fusionaron los dos antiguos contenedores `.resume` independientes en una estructura unificada de una sola página continua.
   - La barra lateral izquierda ahora consolida de forma continua el perfil de contacto, educación, habilidades, competencias y referencias.
   - El panel derecho contiene el perfil profesional, experiencia laboral detallada, certificados y objetivos de aprendizaje.

4. **Corrección Ortográfica, de Redacción y Enlaces:**
   - Se corrigieron acentos faltantes y términos erróneos (como *"Suply Chain"* -> *Supply Chain*, *"Analytic's"* -> *Analytics*, etc.).
   - Se habilitaron enlaces de correo electrónicos cliqueables mediante etiquetas `mailto:`.
   - Se mejoró el formato de los certificados y el aprendizaje continuo estructurándolos en tarjetas cuadriculadas interactivas con efectos de transición suaves (*hover*).
